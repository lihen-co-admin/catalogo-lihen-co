# Responsive y multidispositivo

## Matriz mínima

- 360–430 px: celular.
- 768–1024 px: tableta.
- 1280–1536 px: portátil y escritorio.
- 1920 px o superior: pantalla grande.

## Comprobaciones

Reviso menú, carrusel, tarjetas, botones de mínimo 44 px, modales, filtros, formularios, pie de página y orientación horizontal. No debe existir desplazamiento horizontal accidental.

## Televisor y proyección

Visualizar en TV no equivale a operar con control remoto. Para eventos se recomienda proyectar desde un computador o preparar una interfaz específica con foco visible y navegación direccional.
