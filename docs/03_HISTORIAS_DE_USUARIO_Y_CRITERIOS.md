# Historias de usuario y criterios de aceptación

## HU-01 — Explorar productos

**Como** visitante, **quiero** ver los productos organizados por línea y categoría, **para** encontrar más fácilmente lo que me interesa.

Criterios:

- Se muestran Beauty Care y Style.
- Cada producto tiene nombre, imagen y datos necesarios.
- Los filtros actualizan los productos visibles.
- Existe un mensaje cuando no hay coincidencias.

## HU-02 — Consultar un producto

**Como** cliente, **quiero** abrir el detalle de un producto, **para** revisar información e imágenes antes de consultar.

Criterios:

- El modal abre desde la tarjeta correcta.
- Muestra la galería correspondiente.
- Puede cerrarse con botón, fondo o tecla Escape.
- No se abre contenido perteneciente a otro producto.

## HU-03 — Contactar por WhatsApp

**Como** cliente, **quiero** enviar a LIHEN.CO los productos seleccionados, **para** consultar disponibilidad o realizar mi pedido.

Criterios:

- El mensaje incluye los productos seleccionados.
- El texto se codifica correctamente en la URL.
- El botón abre WhatsApp o WhatsApp Web.
- Si no hay selección, se muestra una orientación clara.

## HU-04 — Registrarme en la inauguración

**Como** participante, **quiero** registrar mi nombre y modalidad, **para** ingresar a la experiencia interactiva.

Criterios:

- Se validan los datos obligatorios.
- No se registra dos veces la misma sesión sin advertencia.
- Se guarda hora y orden.
- Se informa si la sala está cerrada.

## HU-05 — Esperar y jugar mi turno

**Como** participante, **quiero** saber cuándo es mi turno, **para** interactuar solamente cuando me corresponda.

Criterios:

- Se muestra el participante activo.
- Los demás permanecen deshabilitados.
- El cambio de turno se sincroniza.
- La respuesta se acepta una sola vez cuando la regla lo exige.

## HU-06 — Administrar el evento

**Como** administradora, **quiero** controlar juegos, turnos y contingencias, **para** mantener el evento organizado.

Criterios:

- El acceso no está disponible para un visitante común.
- Se puede iniciar y cerrar un juego.
- Se puede activar el siguiente participante.
- Se puede pausar o reanudar el tiempo.
- Las acciones muestran confirmación o error.

## HU-07 — Ver la experiencia en pantalla grande

**Como** espectador, **quiero** visualizar el estado del evento en un televisor o proyector, **para** seguir la actividad sin participar desde esa pantalla.

Criterios:

- La vista no crea participante.
- Los textos y resultados son legibles a distancia.
- La pantalla puede entrar en modo completo.
- Se sincroniza sin controles administrativos visibles.

## HU-08 — Reclamar un premio

**Como** ganador, **quiero** recibir un acceso directo a WhatsApp con un mensaje preparado, **para** solicitar mi recompensa.

Criterios:

- El mensaje identifica el juego o premio.
- El resultado queda registrado.
- No se asigna un premio sin disponibilidad.
- El usuario recibe una confirmación visible.
