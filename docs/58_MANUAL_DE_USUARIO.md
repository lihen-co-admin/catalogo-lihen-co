# Manual de usuario — LIHEN.CO

## Catálogo

1. Abrir la página principal.
2. Usar el buscador o seleccionar Beauty Care/Style.
3. Abrir una tarjeta para ver imágenes, descripción y variantes.
4. Agregar productos a “Mi consulta”.
5. Abrir la selección y preparar un mensaje para WhatsApp.

La página es un catálogo de consulta. No confirma automáticamente inventario, precio final, pago ni entrega.

## Contacto

Completar los campos obligatorios y aceptar la autorización. La página informará claramente si la solicitud fue enviada a Supabase o si quedó guardada solo en el navegador.

## Inauguración

1. Entrar a `/inauguracion/`.
2. Escribir datos reales y un nombre visible apropiado.
3. Elegir modalidad presencial o virtual.
4. Leer y aceptar las reglas.
5. Guardar el código de acceso mostrado.
6. Pulsar “Entrar a la sala”.

## Durante los juegos

- Solo la persona activa responde Trivia, Encuentra la palabra, Reto relámpago y Ruleta.
- Vota tu favorito permite participación simultánea.
- No recargar mientras se envía una respuesta o gira la ruleta.
- Una persona ganadora puede quedar excluida de otros juegos premiados.

## Recompensas

La persona ganadora debe abrir “Ganadores LIHEN”, revisar su código y preparar la reclamación por WhatsApp. La entrega solo queda confirmada cuando LIHEN.CO cambia el estado a “Entregada”.
