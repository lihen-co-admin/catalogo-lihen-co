# ETAPA 27 — Nosotros, promoción y ajustes visuales

## Objetivo
Integrar la página Nosotros con el contenido público corporativo, mejorar la jerarquía visual de carruseles, actualizar iconos de navegación y crear un beneficio de bienvenida consultable por WhatsApp.

## Archivos modificados
- `nosotros.html`: nueva arquitectura de información pública.
- `index.html`: cambio de Bienestar por Así se vive LIHEN.CO.
- `css/storefront.css`: tarjetas compactas, iconografía, página Nosotros, carrusel compacto del drawer y modal promocional.
- `js/storefront.js`: carrusel de recomendaciones del drawer y formulario de bienvenida.
- Páginas HTML principales: sustitución de iconos genéricos del encabezado.

## Información no publicada
No se incorporaron rutas locales, notas de operación, direcciones privadas ni observaciones internas. Solo se empleó contenido corporativo apto para público.

## Pruebas esperadas
1. El carrusel principal avanza automáticamente cada 5 segundos.
2. Todos los carruseles muestran tarjetas de altura controlada.
3. El bloque “¡Esto te va a interesar!” se desplaza horizontalmente.
4. El icono de marca lleva al footer y la bolsa abre Mi selección.
5. La pestaña “10% BIENVENIDA” abre un modal y el formulario prepara una consulta por WhatsApp.
6. Nosotros se adapta a celular, tableta y escritorio.
