# Reporte de validación — Etapa 14

## Validaciones automáticas
- Sintaxis de todos los módulos mediante `npm run check`.
- Archivos nuevos incluidos en el ZIP.
- Sin eventos `onclick` agregados al HTML.
- Una responsabilidad principal por módulo.

## Validaciones que requieren Supabase
- Una persona solo puede votar una vez por ronda.
- Participantes deshabilitados o que salieron no pueden votar.
- Ranking sincronizado entre dispositivos.
- Apertura, cierre y reinicio restringidos a administración.
