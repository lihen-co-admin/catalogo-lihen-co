# Pruebas y validación

## Automatizadas

```bash
npm run check
npm run check:js
```

## Manuales

1. Abrir todas las rutas.
2. Probar menú y enlaces.
3. Buscar y filtrar productos.
4. Abrir y cerrar vista previa.
5. Agregar, incrementar, reducir y quitar productos.
6. Validar contador y persistencia.
7. Comprobar WhatsApp y correo.
8. Revisar páginas legales.
9. Probar teclado, foco y reducción de movimiento.
10. Repetir en celular, tableta y escritorio.

## Evidencia

Para cada prueba registro fecha, dispositivo, navegador, paso ejecutado, resultado esperado, resultado obtenido, captura y corrección.
