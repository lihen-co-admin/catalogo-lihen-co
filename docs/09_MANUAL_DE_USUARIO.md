# Manual de usuario

## Explorar

La persona puede navegar por categorías, usar el buscador y desplazarse por carruseles.

## Vista previa

“Vista previa” abre fotografías e información ampliada. Se puede recorrer la galería y cerrar el modal.

## Mi selección

“Agregar a mi selección” guarda la referencia y muestra una confirmación. En “Mi selección” se ajustan cantidades o se quitan productos.

## Consultar

El botón de WhatsApp prepara un mensaje con las referencias. LIHEN.CO confirma disponibilidad, variantes, precio y condiciones antes del pago.
