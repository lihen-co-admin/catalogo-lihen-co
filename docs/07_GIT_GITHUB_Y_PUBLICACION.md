# Git, GitHub y publicación

## Flujo recomendado

```bash
git status
git add -A
git commit -m "Etapa 35 interacciones y documentación"
git push origin main
```

Después verifico que aparezca `working tree clean` y que GitHub Pages termine el despliegue.

## Revisión pública

Abro la URL en modo incógnito, hago recarga forzada, pruebo rutas internas y compruebo que no existan errores 404. GitHub Pages puede tardar algunos minutos en reflejar el commit.

## Producción

No publico secretos, respaldos con datos personales ni archivos innecesarios. Mantengo una etiqueta o ZIP por versión estable.
